# Statement of Work — Acme Corp Website Redesign

## Project Overview

This document outlines the scope of work for the redesign and development of the Acme Corp marketing website. Work will be completed in three phases over approximately 10 weeks.

**Client:** Acme Corp
**Contractor:** Ian MacDonald
**Rate:** $150/hr
**Payment Terms:** NET 30

---

## Phase 1 — Discovery & Design

- Kickoff meeting and requirements gathering
- Review existing brand assets and style guide
- Audit current website content and structure
- Define site architecture and navigation map
- Wireframes for all key page templates (home, about, services, contact)
- High-fidelity mockups in Figma (desktop and mobile)
- Design review and client sign-off

## Phase 2 — Development

- Set up project repository and development environment
- Implement design system and component library
- Build homepage with hero, features, and testimonials sections
- Build about page with team bios
- Build services page with pricing table
- Build contact page with form and map integration
- Integrate CMS for blog and news posts
- Set up analytics and cookie consent banner
- Cross-browser and responsive testing
- Accessibility audit (WCAG 2.1 AA)

## Phase 3 — Launch & Handover

- Staging environment review with client
- Content migration from old site
- DNS cutover and go-live
- Post-launch smoke testing
- Performance optimisation (Lighthouse score > 90)
- Documentation and CMS training session
- 30-day post-launch support window

---

## Out of Scope

- Custom e-commerce functionality
- Third-party API integrations beyond contact form
- Ongoing hosting or maintenance after support window
